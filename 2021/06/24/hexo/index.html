二、安装hexo、node、git

Hexo官方文档：https://hexo.io/zh-cn/

  1. 安装：npm install hexo-cli -g
  2. 安装node.js： 建议V12.0+
  3. 安装、配置git（需要配置SSH秘钥）

三、创建hexo

  1. 本地文件创建一个文件夹，进入文件夹右键git Bash Here
  2. hexo init 初始化hexo
  3. npm install安装所需要的依赖
  4. hexo server 运行，预览默认端口localhost:4000
